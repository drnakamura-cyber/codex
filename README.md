# 漢方候補サジェスト（参考用）

症状テキストと簡易チェック項目から、漢方薬の候補をルールベースで3つ表示する Next.js アプリです。
**診断や処方を断定するものではありません。**

## 主な仕様

- トップページで症状メモとチェック項目を入力
- 結果ページで候補方剤を3つ表示（理由つき）
- 赤旗症状（胸痛、呼吸困難、意識障害、黒色便、持続高熱）に一致した場合は候補を表示せず、受診案内のみ表示

## 起動方法

```bash
npm install
npm run dev
```

起動後、ブラウザで `http://localhost:3000` を開いてください。

## データとロジック

- 方剤データ: `data/formulas.json`
- スコアリングルール: `lib/rules.ts`

例:
- 冷え → 温める系に加点
- 便秘 → 瀉下・潤腸系に加点
- むくみ → 水分代謝系に加点
